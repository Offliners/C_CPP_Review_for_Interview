# Rotate Array
Detail : [link](https://leetcode.com/problems/rotate-array/)

Given an integer array `nums`, rotate the array to the right by `k` steps, where `k` is non-negative.

### Solution 1 - Modulus
* Time complexity: $O(n)$
* Space complexity: $O(n)$

## Usage
```shell
$ mkdir build && cd build
$ cmake ..
$ make
$ cd ..
$ python3 compare.py
```
