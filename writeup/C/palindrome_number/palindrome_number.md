# Palindrome Number
Detail : [link](https://leetcode.com/problems/palindrome-number/)

Given an integer `x`, return `true` if `x` is a palindrome, and `false` otherwise.

### Solution 1 - Brute Force
* Time complexity: $O(n)$
* Space complexity: $O(n)$


## Usage
```shell
$ mkdir build && cd build
$ cmake ..
$ make
$ cd ..
$ python3 compare.py
```
