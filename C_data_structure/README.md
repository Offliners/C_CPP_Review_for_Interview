# Data Structure in C

## Usage
data_structure_type can be `array`, `linked_list`
```shell
$ cd {data_structure_type}
$ make
$ ./my_{data_structure_type}
```

## Outline
Big-O Cheat Sheet : [Link](https://www.bigocheatsheet.com/)

### Array
* Append: $O(1)$
* Remove: $O(1)$
* Insert: $O(n)$
* Delete: $O(n)$

### Linked List
* Append: $O(n)$
* Remove: $O(n)$
* Insert: $O(n)$
* Delete: $O(n)$
